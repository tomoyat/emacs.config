** default

*** 移動

| cmd   | 役割             |
|-------+------------------|
| C-v   | 次のページ       |
| M-v   | 前のページ       |
| M-f   | 次の単語         |
| M-b   | 前の単語         |
| M-d   | 単語の削除       |
| C-M-a | 次の括弧の終わり |
| C-M-e | 括弧の始まり     |
| C-M-n | 関数の終わり     |
| C-M-p | 関数の始まり     |
| M-g g | line jump        |

*** その他

| C-h b        | key bind一覧                            |
| C-x ^        | 縦に広げる                              |
| C-x }        | 横に広げる                              |
| C-x {        | 横に広げる                              |
| C-u 5        | そのあとのコマンドをくりかえす          |
| C-x C-b d+ x | C-x C-bの画面dで選択したbufferをxで消す |
